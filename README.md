# 進階股票分析與多指標回測系統 📈🧠
(這是專案是由AI生成的，我只是負責提供想法與修改)

本專案是一個整合了 **深度強化學習 (DRL)**、**LSTM 神經網路** 與 **多重技術指標** 的進階股票分析儀表板。系統旨在透過現代 AI 技術與傳統量化策略的結合，提供全方位的盤勢解析與回測工具。

## 🌟 核心功能

### 1. AI 智能大腦 (DRL-LSTM)
- **最新架構**：使用 2 層 LSTM、128 個神經元的深度模型，優化時間序列特徵提取。
- **強化學習**：基於 Stable Baselines3 PPO 演算法，專注於捕捉多頭市場機會。
- **宏觀感知**：具備 60 天、20 天及 **252 天 (1年)** 的宏觀位階判讀能力。

### 2. 多維指標綜合策略
- **綜合共振 (Tri-Core Resonance)**：融合 MACD、RSI 與自創 DMPI 指標的動態切換策略。
- **自創 DMPI (動態市場壓力指數)**：結合價格壓力、成交量與 ATR 波動的趨勢領先指標。
- **傳統經典指標**：提供 RSI (相對強弱) 與 MACD (平滑異同移動平均) 的即時與回測分析。

### 3. 動態視覺化儀表板
- **互動 K 線圖**：自動標註策略買賣點位。
- **即時價格資訊**：頂部顯示最新收盤價、當日高低點與漲跌幅變色提醒。
- **詳細回測報告**：一鍵生成四大策略的總報酬率、MDD (最大回撤)、勝率及近期交易明細。
- **基本面剖析**：整合公司基本資料與年度財報摘要。

## 🚀 快速啟動

### 環境需求
請確保已安裝 Python 3.10+，並執行以下指令安裝必要套件：
```bash
pip install -r requirements.txt
```

### 執行程式
雙擊或在終端機執行：
```powershell
./run.bat
```

## 📂 專案架構
- `main.py`: 系統入口與錄 GUI (CustomTkinter) 邏輯。
- `ml_engine.py`: AI 推論引擎，負責載入 DRL 模型與特徵處理。
- `backtester.py`: 交易策略回測核心。
- `strategy.py`: 技術指標生成與策略邏輯。
- `data_loader.py`: 設定 yfinance 資料抓取。
- `chart.py`: 負責互動式圖表繪製。
- `pretrained_brain/`: 存放預訓練好的 DRL 模型權重。

## 🛠️ 技術棧
- **GUI**: CustomTkinter (現代化暗色介面)
- **Data**: Pandas, yfinance
- **AI/ML**: PyTorch, Stable Baselines3 (PPO), Gymnasium
- **Charting**: Matplotlib, Mplfinance
