# Finance Buyable Scan (台股精銳掃描)

本技能用於執行台灣股市的自動化掃描、評分排名以及投資組合狀態檢查。它採用「綜合共振」策略，透過 MACD、RSI 與 DMPI 指標動態判斷多、空或盤整格局。

## 主要功能
1. **全市場掃描**：分析約 150 支台股精銳個股，尋找符合「綜合共振」條件的買進或賣出點。
2. **評分排名**：針對出現買進訊號的股票，依據趨勢強度 (Regime)、RSI 強勢度及成交量進行加權評分 (0-100+)。
3. **持股檢查**：自動比對目前的投資組合 (Portfolio)，確認是否有已持有的股票發出「賣出」警示。
4. **訊息整合**：將掃描結果、排名與警示彙整，並可透過 Discord 接口發送報告。

## 使用方式

### 執行掃描
你可以直接執行主腳本來獲取完整的 JSON 或格式化報表：

```bash
# 在 Linux / WSL 環境下執行
./daily_stock_scan.sh
```

或是手動執行 Python 評分模組：
```bash
# 針對特定股票進行排名評分
python3 daily_pick.py 2330.TW 2317.TW 2454.TW
```

## 依賴模組
- `yfinance`: 獲取市場數據。
- `pandas`: 數據處理與技術指標計算。
- `strategy.py`: 核心策略大腦（包含 DMPI, RSI, MACD 邏輯）。
- `data_loader.py`: (建議) 處理網路連線與 SSL 負載。

## 策略邏輯摘要
- **多頭通道**：DMPI 在 -20 到 30 之間。
- **空頭搶短**：DMPI 在 -40 到 3 之間。
- **盤整回歸**：RSI < 50 買進，RSI > 80 賣出。
- **停利/停損**：動態偵測 DMPI 觸及邊界或 MACD 趨勢轉折時發出出場訊號。
